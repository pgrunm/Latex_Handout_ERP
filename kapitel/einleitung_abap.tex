\section{Was ist ABAP?}

\begin{itemize}
    \item Programmiersprache entwickelt von SAP.
    \item ABAP stand ursprünglich für Allgemeiner BerichtsAufbereitungs Prozessor.
    \item Heutige Bedeutung: Advanced Business Application Programming.
    \item Entwicklung begann in den 1980er Jahren bis heute.
    \item Sprache der vierten Generation. % Was bedeutet das konkret. Wenn Zeit ist gerne ergänzen..
    \item IDE wird direkt mitgliefert.
    \item Native Objektorientierung seit 1999.
\end{itemize}
